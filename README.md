# Budget Assistant LLM

This Space deploys a Streamlit budgeting prototype with a local LLM-based "AI What-If Planner" running through Ollama.

Main features:
- data-driven budget forecasting from transaction history
- priority-aware budget reallocation
- local LLM interpretation of natural-language spending scenarios

Example scenario:
> Reduce Eating out by 40% and Shopping by 30%. Keep Savings unchanged.
